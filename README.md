# Controle de Itens Perdidos

Projeto simples criado para treinar lógica de programação e organização de informações.

A ideia é registrar itens perdidos ou encontrados, anotando informações básicas para controle.

## Funcionalidades
- Registrar um item (nome, local e data)
- Listar itens cadastrados
- Remover itens da lista

## Tecnologias
- JavaScript
- Conceitos básicos de lógica

## Status
Projeto em desenvolvimento.
